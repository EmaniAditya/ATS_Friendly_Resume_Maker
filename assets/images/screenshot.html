How to capture a screenshot:

  1. Open the application in your browser
  2. Ensure the form is populated with sample data
  3. Make sure the resume preview is visible
  4. Use browser developer tools to capture a full-page screenshot
  5. Crop the image to show both the form and preview
  6. Save the image as "app-screenshot.png" in this directory
  7. Replace this file with the actual screenshot
